# 🔍 Comparateur de textes touristiques

Cette application Streamlit permet de comparer deux descriptions d’activités touristiques afin de mesurer leur similarité sémantique et d’identifier leurs différences.

Elle est utile pour :
- comparer des offres touristiques
- détecter des contenus similaires
- analyser des descriptions marketing
- améliorer la qualité de rédaction

---

# 🚀 Fonctionnalités

- 📊 Score global de similarité (0 à 100%)
- 🧠 Analyse sémantique des phrases
- 🔍 Comparaison phrase par phrase
- 🟢🟠🔴 Mise en évidence des différences
- ⚡ Interface simple et rapide avec Streamlit

---

# 🧠 Technologie utilisée

- Python
- Streamlit
- SentenceTransformers (modèle NLP)
- NumPy
